# Requirements: KPI Light

**Defined:** 2026-04-12
**Core Value:** Upload a data file and immediately see sales/revenue KPIs visualized on a dashboard — zero friction from raw data to insight.

## v1.5 Requirements

Requirements for v1.5 Segmented Controls. Each maps to roadmap phases.

### Segmented Control

- [x] **SEG-01**: Reusable SegmentedControl component with pill-shaped container, rounded ends, dark active state, light inactive state
- [x] **SEG-02**: Sales/HR tab navigation rendered as segmented control in navbar
- [x] **SEG-03**: Date range presets rendered as segmented control in SubHeader
- [x] **SEG-04**: Balken/Linie chart type toggle rendered as segmented control
- [x] **SEG-05**: DE/EN language toggle rendered as segmented control in navbar
- [x] **SEG-06**: Full DE/EN i18n parity maintained

## Future Requirements

None deferred from this milestone.

## Out of Scope

| Feature | Reason |
|---------|--------|
| Custom theme colors for segmented control | Uses existing semantic color tokens; no new theming needed |
| Animated transitions between active states | Keep it simple — instant state switch matches screenshot |
| Keyboard navigation (arrow keys) | Internal tool, mouse/touch sufficient for v1.5 |
