    **1. backend/app/routers/__init__.py** — Empty init file.

    **2. backend/app/routers/uploads.py** — Create the upload router:

    ```python
    from datetime import datetime, timezone
    from fastapi import APIRouter, Depends, HTTPException, UploadFile
    from sqlalchemy import select, func
    from sqlalchemy.dialects.postgresql import insert as pg_insert
    from sqlalchemy.ext.asyncio import AsyncSession

    from app.database import get_async_db_session
    from app.models import SalesRecord, UploadBatch
    from app.parsing.erp_parser import parse_erp_file
    from app.schemas import UploadBatchSummary, UploadResponse, ValidationErrorDetail

    router = APIRouter(prefix="/api")
    ```

    **POST /api/upload** endpoint:
    - Accept `file: UploadFile` parameter.
    - Validate filename extension: if not `.csv` or `.txt` (case-insensitive), raise `HTTPException(status_code=422, detail=f"Unsupported file type: {file.filename}. Only .csv and .txt files are accepted.")` — per D-07, UPLD-02.
    - Read file contents: `contents = await file.read()`.
    - Call `parse_erp_file(contents, file.filename)` to get `(valid_rows, errors)`.
    - Determine status per D-11: `"success"` if no errors, `"partial"` if both valid_rows and errors, `"failed"` if no valid_rows (either all failed validation or file-level error).
    - Create UploadBatch record: `batch = UploadBatch(filename=file.filename, uploaded_at=datetime.now(timezone.utc), row_count=len(valid_rows), error_count=len(errors), status=status)`. Add to session, flush to get batch.id.
    - If valid_rows is not empty: set `upload_batch_id` on each row dict to `batch.id`. Use `pg_insert(SalesRecord).values(valid_rows).on_conflict_do_nothing(index_elements=["order_number"])` per D-03. Execute the statement. After execution, update batch.row_count to the actual number of inserted rows using `result.rowcount` (since ON CONFLICT DO NOTHING may skip duplicates).
    - Commit the session.
    - Return `UploadResponse(id=batch.id, filename=batch.filename, row_count=batch.row_count, error_count=batch.error_count, status=batch.status, errors=[ValidationErrorDetail(**e) for e in errors])`.

    **GET /api/uploads** endpoint:
    - Accept `db: AsyncSession = Depends(get_async_db_session)`.
    - Query: `result = await db.execute(select(UploadBatch).order_by(UploadBatch.uploaded_at.desc()))`.
    - Return `[UploadBatchSummary.model_validate(batch) for batch in result.scalars().all()]` with `response_model=list[UploadBatchSummary]`.

    **DELETE /api/uploads/{batch_id}** endpoint:
    - Accept `batch_id: int` path parameter and `db: AsyncSession`.
    - Query batch: `batch = await db.get(UploadBatch, batch_id)`.
    - If not found, raise `HTTPException(status_code=404, detail="Upload batch not found")`.
    - Delete: `await db.delete(batch)` — cascade deletes associated SalesRecords per the model relationship.
    - Commit.
    - Return `{"detail": "deleted", "id": batch_id}`.

    **3. backend/app/main.py** — Add router import and include:
    - Add `from app.routers.uploads import router as uploads_router` after existing imports.
    - Add `app.include_router(uploads_router)` after the app instantiation.
    - Keep the existing `/health` endpoint unchanged.
