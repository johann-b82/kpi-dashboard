(2) for each date column in row, try parse_german_date — if None and value non-empty, add error `{"row": row_num, "column": col_name, "message": f"unparseable date '{value}'"}`.     - Function `row_to_dict(row: dict[str, str]) -> dict`: Convert a validated row dict to DB-ready dict.       5. Return (valid_rows, errors).
