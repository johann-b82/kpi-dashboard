**Setup (once):** Confirm `sample_export.csv` data is loaded — the dashboard should already show non-zero KPI values. If it does not, upload it via the Upload page.

**The 4 × 2 matrix (D-18):** cycle every combination below. For each row, verify every bullet. Mark PASS or note the defect.

Languages: **DE** (set via NavBar LanguageToggle → German), **EN** (set via LanguageToggle → English).
Presets: **Diesen Monat / This month**, **Dieses Quartal / This quarter**, **Dieses Jahr / This year**, **Gesamter Zeitraum / All time**.

For EACH of the 8 cells:

1. **All 3 KPI cards render** (Total revenue / Average order value / Total orders) — uniform height, no layout break.

2. **Both delta badges appear to the right of the KPI value** (vs. Vorperiode + vs. Vorjahr) — or both em-dashes for Gesamter Zeitraum and Dieses Jahr's vs. Vorjahr baseline where data does not extend back.

3. **Delta badge format is locale-correct:**
   - DE: `▲ +12,4 %` (comma decimal, space before %, NBSP)
   - EN: `▲ +12.4%` (period decimal, no space before %)
   - Up-triangle ▲ for positive in primary color; down-triangle ▼ for negative in destructive red.

4. **Secondary label under each badge is contextual to the preset:**
   - Diesen Monat → "vs. März" (DE) / "vs. March" (EN) — actual current month's prior month name
   - Dieses Quartal → "vs. Q1" / "vs. Q2" etc. (locale-invariant)
   - Dieses Jahr → "vs. 2025" for the vsYear badge; "—" (em-dash) for the vsPeriod badge (thisYear preset has no prev-period baseline per D-04)
   - Gesamter Zeitraum → "—" on BOTH badges in both languages

5. **Em-dash hover tooltip:** Hover any em-dash badge. DE tooltip: "Kein Vergleichszeitraum verfügbar". EN tooltip: "No comparison period available".

6. **Chart overlay:** RevenueChart shows a blue current series + an amber prior-period ghost series at reduced opacity. Both series share the X axis. For Gesamter Zeitraum, the prior series is absent (not rendered).

7. **Chart legend labels:**
   - Diesen Monat → "Umsatz April" / "Umsatz März" (DE) or "Revenue April" / "Revenue March" (EN) — actual months
   - Dieses Quartal → "Umsatz Q2" / "Umsatz Q1"
   - Dieses Jahr → "Umsatz 2026" / "Umsatz 2025"
   - Gesamter Zeitraum → "Umsatz" / "" (only current series)

8. **Language switch live re-render:** With a non-allTime preset selected, click LanguageToggle to flip DE ↔ EN. ALL of: card delta badges, secondary labels, tooltip copy, chart legend entries must update IMMEDIATELY without a page refresh. No stale DE-while-EN fragments. Nav items + page copy also swap (baseline Phase 7 behavior).

9. **German tone check:** Read the DE strings out loud. Confirm:
    - Tooltip reads as natural informal German (`"Kein Vergleichszeitraum verfügbar"` — acceptable; not "Sie"-form; not stiff).
    - `"vs. März"`, `"vs. Vorperiode"`, `"Umsatz April"`, `"Umsatz 2026"` all read cleanly.
    - Short-range custom (if you navigate to a custom 3-day range — optional) reads `"vs. 3 Tage zuvor"`.

10. **No drift between cards and chart:** For any preset with non-null baselines, the direction (positive/negative) of each card's delta matches the visual direction of the chart overlay (current series above vs below the prior ghost series).

**Additional edge check — Gesamter Zeitraum em-dash uniformity (SC5 explicit requirement):** In both DE and EN with Gesamter Zeitraum selected, confirm all SIX badge slots (3 cards × 2 badges) show em-dashes. No stray numeric values.
