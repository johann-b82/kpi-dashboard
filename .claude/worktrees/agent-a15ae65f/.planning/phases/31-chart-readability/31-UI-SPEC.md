## Component Contracts

### 1. XAxis tick formatter ‚Äî `formatMonthYear`

Location: `src/lib/chartTimeUtils.ts` (new shared utility)

Contract:
- Input: ISO date string (`YYYY-MM-DD`) or `YYYY-MM` (HR charts ‚Äî append `-01` before parsing)
- Output: `MMM 'YY` string using `Intl.DateTimeFormat` with `month: "short"` + 2-digit year slice
- Locale-aware: receives `locale` string from i18n context (`"de"` / `"en-US"`)
- Applied in: `RevenueChart.tsx` `formatXAxis` fallthrough (non-`thisMonth` branches) and `HrKpiCharts.tsx` `MiniChart.formatMonth`
- Do NOT apply to `thisMonth` preset weekly bucket labels

### 2. Month spine ‚Äî `buildMonthSpine` + `mergeIntoSpine`

Location: `src/lib/chartTimeUtils.ts`

Contract:
- `buildMonthSpine(start, end)`: returns ordered `YYYY-MM-01` strings covering `[start, end]` inclusive
- `mergeIntoSpine(spine, points)`: left-join API `ChartPoint[]` into spine; missing months ‚Üí `revenue: null`
- `allTime` edge case: when `startDate` or `endDate` is `undefined`, derive bounds from first/last element of API response; return empty spine if API returns zero points
- Applied to: Sales/Revenue chart only ‚Äî HR chart data is already dense (12 months always present)

### 3. Year boundary `ReferenceLine`

Location: inline JSX in `RevenueChart.tsx` and `HrKpiCharts.tsx` MiniChart

Contract:
- Rendered for each date where `date.slice(5, 7) === "01"` (January) present in the spine
- `stroke="var(--color-border)"`, `strokeDasharray="4 2"`, `strokeWidth={1}`
- Label: `{ value: date.slice(0, 4), position: "insideTopLeft", fontSize: 10, fill: "var(--color-muted-foreground)" }`
- `x` value must match exact `dataKey="date"` format (`YYYY-MM-01`) ‚Äî string equality match required
- Guard: only render when `yearBoundaryDates(spine).length > 0` (skip for single-year or `thisMonth` views)

### 4. XAxis explicit `ticks` prop

Location: `RevenueChart.tsx` `XAxis` component

Contract:
- When gap-fill spine is active, pass `ticks={spine}` explicitly to `XAxis`
- Prevents Recharts from skipping null-revenue month tick marks
- HR chart does not need this ‚Äî its data is always dense
